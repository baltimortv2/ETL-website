# ETL Website

Современный веб-сайт для ETL решений с красивым дизайном и анимациями.

## Особенности

- 🎨 Современный дизайн с градиентами
- 🌙 Переключение между темной и светлой темой
- ⚡ Анимации на основе Framer Motion
- 📱 Адаптивный дизайн
- 🚀 Быстрая загрузка с Vite
- 🎯 Автоматический деплой на GitHub Pages

## Технологии

- React 18
- Vite
- Framer Motion
- Lucide React
- CSS3

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Деплой на GitHub Pages
npm run deploy
```

## Деплой

Сайт автоматически деплоится на GitHub Pages при пуше в main ветку.

URL: https://baltimortv2.github.io/ETL-website/
